### MyBatis
SqlSessionFactory - DefaultSqlSessionFactory<br />依赖 DataSource<br />Session = Connection<br />DataSource 创建 Connnection

两种可能：

- SqlSessionFactory 与 DataSourceTransactionManager 使用不同的 DataSource
- 相同的 DataSource 会不会调用两次 getConnection()，可能会返回不同的  Connection

均会返回不同的 Connection，开启不同的事务

解决方案：<br />**SqlSessionFactory 与 DataSourceTransactionManager 使用相同的 DataSource，并且在开启 Session 返回相同的 Connection**


TransactionFactory 实现：

- JdbcTransactionFactory - Connection getConnection() 会调用一次 DataSource#getConnection() 方法
- ManagedTransactionFactory -  Connection getConnection() 会调用一次 DataSource#getConnection() 方法
- SpringManagedTransactionFactory
